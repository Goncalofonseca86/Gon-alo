# leirisonda2025

Gestor de obras e manutenção de piscinas da Leirisonda.

## Descrição

Sistema web para gestão de obras e manutenção de piscinas, desenvolvido para a Leirisonda.

## Tecnologias

- React 18
- Vite
- JavaScript/JSX

## Instalação

```bash
npm install
```

## Desenvolvimento

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Preview

```bash
npm run preview
```

## Deploy

```bash
npm run serve
```

## Estrutura do Projeto

```
src/
├── contexts/
│   └── AuthContext.jsx
├── App.jsx
├── index.css
└── main.jsx
```

## Funcionalidades

- Sistema de autenticação
- Gestão de obras
- Manutenção de piscinas
- Interface responsiva
